## Installation

```bash
npx devcn-ui add sonner
```

## Usage

```tsx
import { Sonner } from '@repo/shadcn-ui/components/sonner';

export default function Example() {
  return (
    <Sonner>
      Your content here
    </Sonner>
  );
}
```

| `children` | `ReactNode` | - | The content of the component |
